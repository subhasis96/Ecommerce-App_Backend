# Ecommerce-App_Backend
A fully functional e-commerce platform that enables users to browse products, manage carts, place orders, and complete secure payments. The project includes both customer-facing features and admin-level management tools, providing a complete end-to-end online shopping experience.

Features:
--View detailed product information
--Add or remove items from cart
--Add, update, delete products
--Manage product categories
--Inventory management

Tech-Stack:
--Java 17
--Spring Boot
--JWT based authentication
--RestAPIs
